# Accounts Payable Vendor Compliance Dashboard

## Project Overview
Excel dashboard for monitoring vendor payments, overdue invoices, liabilities and compliance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

## Key Metrics
- Total Invoices: 800
- Total Liability: ₹20,87,522
- Overdue Invoices: 543
- Overdue Amount: ₹14,18,926
- Paid Amount: ₹7,50,479

## Key Insight
68% of total liability is overdue and BluePrints is the highest-risk vendor.
